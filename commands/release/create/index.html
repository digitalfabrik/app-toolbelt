    create

    creates a new release for the specified platform

    Usage

    app-toolbelt v0 release create <platform> <new-version-name> <new-version-code> [options]

    Arguments

    Argument Description
    <platform>
    <new-version-name>
    <new-version-code>

    Options

    Option Description
    --production-release Whether this is a production or a pre-release. (default: false)
    --release-notes <release-notes> The release notes as JSON string, will be auto-generated otherwise.
    --hotfix Generate release notes only from changes since the branch was cut. (default: false)
    --branch <branch> The current branch, required when --hotfix is set.
    --private-key <private-key> Github app private key as base64 pem format
    --owner <owner> Github owner (e.g. "digitalfabrik")
    --repo <repo> Github repository
    --github-app <github-app> Github app id (default: 59249)
